# projeto-PI-4semestre
Sistema Operacional Ultragaz

descrição:
O Sistema Operacional Ultragaz tem como objetivo integrar tecnologia de forma simples e eficiente à empresa Ultragaz, substituindo métodos ultrapassados de registro de dados de vendas e controle de estoque. Com essa modernização, buscamos otimizar processos e melhorar a gestão das informações, proporcionando um ambiente mais ágil e produtivo.

instalação:
Para instalar o projeto, siga as etapas abaixo:
1. No terminal do seu editor de código, execute o comando: 
git clone https://github.com/Ray-martins1428/projeto-PI-4semestre

2. Após clonar o repositório, navegue até a pasta do projeto com:
cd projeto-PI-4semestre

3. Em seguida, instale as dependências necessárias utilizando:
npm install

4. Crie um arquivo .env e adicione as variáveis de ambiente necessárias para conectar ao banco de dados.

uso:
Para iniciar o projeto e verificar seu funcionamento, execute no terminal (certifique-se de que o caminho do terminal está correto):
npm start
